# Love2D Project Generator

A small application that lets you quickly create new Love2D projects using predefined templates and optional libraries.

## Features

* Create new projects in seconds
* Use custom project templates
* Automatically copy selected libraries
* Save frequently used paths so you don't need to retype them

## How to Use

1. Launch the application
2. Enter the destination folder for the project
3. Enter the project name
4. Select a template
5. (Optional) choose libraries to include
6. Click **Create Project**
